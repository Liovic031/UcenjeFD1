# Ucenje Frontend Development 📚

This repository contains all my exercises and projects while learning Frontend Development at Edunova.  
It includes HTML, CSS, JavaScript and React practice projects, focusing on building layouts, interactivity, and basic web apps.

## 📚 Technologies Covered

- **Structure:** Semantic HTML5  
- **Styling:** CSS3 (Flexbox, Grid, Responsive Design), Bootstrap v5, Foundation CSS, Tailwind CSS 
- **Logic:** JavaScript basics and DOM manipulation (algorithmic exercises)  
- **Framework:** React.js
- **Tools:** Git, GitHub, FileZilla, VS code
  
## 🔗 Access the projects

- Exercises are organized by topic (HTML, CSS, JS)
- Open `index.html` files in your browser to try each project
- or check https://edunova5.origo.hr/
